# PureView 使用说明

## 打开视频

- 双击打开软件后，点击左侧「添加文件夹」扫描本地视频
- 或直接将视频文件拖入窗口播放

## 播放控制

| 操作 | 方式 |
|------|------|
| 播放 / 暂停 | 按 `空格键` 或点击播放按钮 |
| 快进 / 快退 | `← →` 1秒，`Ctrl+← →` 3秒 |
| 调节音量 | `Shift+↑↓` 或拖动音量滑块 |
| 静音 | 按 `M` |
| 调节倍速 | `]` 加速，`[` 减速，`Backspace` 恢复原速 |
| 全屏 | 按 `F`，按 `Esc` 退出 |

## 关键帧书签

在播放过程中标记重要时间点，方便日后快速跳转。

- 按 `K` 添加书签，`Shift+K` 删除
- 按 `↑↓` 在书签之间跳转
- 书签会显示在进度条上，自动保存

## 播放列表

在视频库页面点击「新建播放列表」，可将视频分组管理。

## 自定义快捷键

所有快捷键均可在设置页面中修改，支持冲突检测和一键恢复默认。

## 支持格式

MP4、MKV、AVI、MOV、WMV、FLV、WebM、M4V、TS、M2TS、MPG、MPEG、RM、RMVB、3GP
